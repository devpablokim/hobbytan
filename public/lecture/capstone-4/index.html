PROMPT ENGINEERING

프롬프트 설계의 핵심: Co-STAR

  • Context — 배경/맥락을 제공
  • Objective — 목표를 명확히
  • Style — 원하는 톤/스타일
  • Tone — 감정/분위기
  • Audience — 대상 독자
  • Response — 출력 형식

💡 예시

Context: 대학생 3명이 친환경 텀블러를 캡스톤으로 기획
Objective: 시장 분석 보고서 초안 작성
Style: 전문 컨설팅 보고서 형식
Tone: 객관적, 데이터 기반
Audience: 심사위원 교수진
Response: 마크다운 표 포함 3페이지

  • 2개 이상 AI에서 교차 확인
  • 최신 데이터는 웹 검색 모드 사용
