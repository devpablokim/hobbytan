사용자 데이터 삭제 안내

하비탄AI(HOBBYTAN AI)와 AI조씨(AIJOSSI)는 이용자의 개인정보 삭제 요청을 존중합니다. Facebook 또는 기타 소셜 로그인을 통해 수집된 데이터를 포함하여, 아래 방법으로 데이터 삭제를 요청할 수 있습니다.

📋 Facebook 데이터 삭제 콜백 안내
Facebook 로그인으로 수집된 데이터는 Facebook의 데이터 삭제 요청 정책에 따라 처리됩니다. 아래 양식을 통해 삭제를 요청하시면 확인 코드를 발급해 드립니다.

삭제 대상 데이터

  • 계정 정보 (이메일, 이름, 프로필 사진)
  • 소셜 로그인 연동 정보 (Facebook, Google, Instagram 등)
  • 서비스 이용 기록 (교육 수강 기록, 워크샵 참가 기록)
  • 사용자 생성 콘텐츠 (게시물, 댓글)
  • 쿠키 및 접속 로그

삭제 요청 방법

  1. 아래 양식에 가입 시 사용한 이메일 주소를 입력합니다.
  2. "데이터 삭제 요청" 버튼을 클릭합니다.
  3. 확인 코드가 발급됩니다. 이 코드를 보관하세요.
  4. 요청 접수 후 30일 이내에 모든 데이터가 영구 삭제됩니다.
  5. 삭제 완료 시 등록된 이메일로 통보합니다.

Data Subject to Deletion

  • Account information (email, name, profile photo)

  • User-generated content (posts, comments)

How to Request Deletion

  1. Enter the email address used during registration in the form below.

  3. A confirmation code will be issued. Please keep this code for your records.
  4. All data will be permanently deleted within 30 days of the request.
  5. You will be notified by email upon completion.
