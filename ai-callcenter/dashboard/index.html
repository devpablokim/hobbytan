한국 기준 AI 콜센터 구축 절차

구축 절차 (6단계)

1
ElevenLabs 가입 + 에이전트 생성
elevenlabs.io → Creator 이상 플랜 → Conversational AI → Create Agent
2
Twilio 가입 + 한국 번호 발급
twilio.com → 사업자등록증 제출 → 한국 번호(070/02) 구매 (또는 US 번호로 테스트)
3
ElevenLabs ↔ Twilio 연동
ElevenLabs Agent → Phone Numbers → Connect Twilio → Account SID + Auth Token 입력
4
응답 룰 + 시스템 프롬프트 설정
FAQ 룰 입력 + Knowledge Base 문서 업로드 + 상담원 연결 트리거 설정
5
테스트 통화
발급받은 번호로 직접 전화 → AI 인사말 확인 → FAQ 질문 → 상담원 연결 테스트
6
실 운영 전환
한국 번호 정식 발급 → 운영 모니터링 대시보드 연동 → 고객 안내
