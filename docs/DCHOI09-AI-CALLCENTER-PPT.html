01

서비스 개요 — 24시간 AI 상담

🎤

음성 상담

AI가 전화를 받아 자연스러운 음성으로 대화합니다. 제품 안내, 프로모션, 추천까지 자동 처리.

💬

텍스트 상담

채팅으로 빠르고 정확한 안내. 음성 안내가 필요하면 버튼 하나로 전환.

📊

관리 대시보드

모든 상담 기록을 한눈에. 고객 정보, 상담 유형, 에스컬레이션 현황 실시간 확인.

핵심 원리: 정해진 룰에 해당하는 질문은 AI가 즉시 응답 (80%+ 처리). 복잡하거나 룰에 없는 건은 담당자에게 즉시 연결합니다.

02

음성 상담 — AI가 전화를 받습니다

  • ElevenLabs Conversational AI 기반 — 자연스러운 한국어 음성 대화
  • 제품 카탈로그 자동 안내 (제품명, 가격, 특징, 성분)
  • 진행 중인 프로모션 자동 안내
  • 고객 연령에 따른 맞춤 제품 추천
  • 실시간 트랜스크립트 — 대화 내용이 화면에 실시간 표시
  • 룰에 없는 질문 → "담당자 Monica에게 연결해 드리겠습니다"
음성 상담 URL
hobbytan-ai.web.app/dchoi09-callcenter/
03

텍스트 상담 — 채팅 + 음성 토글

인테이크 폼

상담 시작 전 4가지 정보를 수집하여 개인화된 상담을 제공합니다:

  • 성함 (필수) — "ㅇㅇㅇ 고객님"으로 호칭
  • 전화번호 (필수)
  • 이메일 (필수)
  • 거주 지역 (선택) — 미국 거주 시 입력

채팅 기능

  • 텍스트 입력 → AI 즉시 응답
  • 🔊 음성 안내 토글 — AI 답변을 음성으로 읽어줌
  • 대화 기록 자동 저장
  • 이전 상담 이력 확인 가능
텍스트 상담 URL
hobbytan-ai.web.app/dchoi09-callcenter/chat/
04

개인화 상담 — 이름을 불러주는 AI

"김철수 고객님, 무엇을 도와드릴까요?"

인테이크 폼에서 입력한 이름으로 자연스럽게 호칭합니다

  • 폼 입력 → AI 프롬프트에 고객 이름 자동 주입
  • 대화 전체에서 "ㅇㅇㅇ 고객님" 형태로 일관된 호칭
  • 수집된 고객 정보(이름, 연락처, 이메일, 지역)는 대시보드에 자동 기록
  • 재방문 시 상담 이력과 고객 정보 즉시 확인 가능
